# Stable Tracker

Stable Tracker 是一个基于 ByteTrack 的实时目标跟踪器，专门设计用于视频监控和行为分析等应用场景。该系统具有高效、稳定的特点，能够满足实际业务需求。

## 主要功能

### 行人身份特征向量注册

通过以下命令执行特征向量注册流程：

```python
python register_main.py
```

### 实时身份识别

运行以下命令启动身份识别功能：

```python
python match_main.py
```
